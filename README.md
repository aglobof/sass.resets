# Sass Resets

> A glob of Sass resets.

## Install

With [Bower](http://bower.io) - `bower install aglobof-sass-resets`

Import from your root sass file.

```
// Sass Root File
// ----------

// Import defaults
@import "/path/to/bower/sass-defaults/normalize";
@import "/path/to/bower/sass-defaults/box";
```
